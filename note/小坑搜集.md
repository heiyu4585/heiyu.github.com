## "/n"在html中如何正确显示
 
####  替换回车换行
 
```
    //替换所有的回车换行
    function TransferString(content)
    {
        var string = content;
        try{
            string=string.replace(/\r\n/g,"<br>")
            string=string.replace(/\n/g,"<br>");
        }catch(e) {
            alert(e.message);
        }
        return string;
    }
```
